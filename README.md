# Blackjack Game (C)

A simple command-line Blackjack game written in **C** as part of my **first-semester programming assignment**.

## Overview

This project is a text-based implementation of the classic Blackjack card game. It was developed to practice fundamental C programming concepts, including variables, loops, conditional statements, functions, and basic game logic.

## Features

- Command-line interface (CLI)
- Player vs. Dealer gameplay
- Hit and Stand actions
- Random card generation
- Blackjack score calculation
- Win/Lose/Draw determination

## Technologies

- **Language:** C
- **Compiler:** GCC (or any standard C compiler)

## Getting Started

### Clone the repository

```bash
git clone https://github.com/Zahin75/Assignment-First-Semester-.git
cd Assignment-First-Semester-
```

### Compile

```bash
gcc *.c -o blackjack
```

or

```bash
gcc main.c -o blackjack
```

### Run

**Linux/macOS**
```bash
./blackjack
```

**Windows**
```bash
blackjack.exe
```

This project was created during my **first semester** as a university programming assignment. Its purpose was to strengthen my understanding of C programming fundamentals by building a simple command-line game.
